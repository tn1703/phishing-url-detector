# Phishing URL Detection

## Project Overview
This project detects potentially malicious phishing URLs using machine learning techniques.

## Features
- URL safety prediction
- Dataset-based phishing analysis
- Machine learning classification
- Simple interface for URL checking

## Technologies Used
- Python
- Jupyter Notebook
- Machine Learning
- Pandas
- Scikit-learn

## Project Purpose
This project was adapted from an open-source learning project and customized for portfolio and educational purposes.

## Outcome
The system analyzes URLs and predicts whether they may be phishing or safe.
